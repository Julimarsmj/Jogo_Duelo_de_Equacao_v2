# Jogo_Duelo_de_Equacao_v2
Desenvolvimento de um projeto prático em Java/Swing (NetBeans) para demonstrar e ensinar, de forma dinâmica, como estabelecer e manipular a conexão com um banco de dados relacional (como o MySQL) através de um jogo interativo. Esta é a segunda versão do projeto, que inclui correções de bugs da versão anterior e melhorias nas mecânicas de jogo.
